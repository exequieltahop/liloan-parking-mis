## Overview

This system is for tracking parking slot with iot (nodemcu and laravel api)
using the ultrasonic sensor to set the status if free or not.

## Requirements

- php 8.2+
- node.js
- mysql
- composer
- git
- vscode (any editor)

## installation

- git clone this repo
- cd the project
- open the terminal (root dir)
- type composer install
- copy and paste .env.example in the root dir
- rename it into .env
- set up database
- type in the terminal php artisan key:generate
- type php artisan serve
